# XOR Modifier (Qt/C++)
Тестовое приложение для пакетной обработки файлов с применением битовой операции XOR. Разработано на C++ с использованием фреймворка Qt.

## 📋 Функциональные возможности
### Настройки пользователя
- **Маска входных файлов** — фильтр по расширению или конкретному имени (например, `*.txt`, `data.bin`)
- **Удаление исходников** — опциональное удаление входных файлов после обработки
- **Путь сохранения** — директория для результирующих файлов
- **Конфликт имён** — выбор поведения при совпадении:
  - Перезапись существующего файла
  - Автоматическая модификация имени (добавление числового счётчика)
- **Режим работы**:
  - Однократный запуск
  - Периодический запуск по таймеру
- **XOR-value** — 8-байтное значение для модификации данных

## 🛠 Технологии
- **Язык:** C++17
- **Фреймворк**: Qt (версия 5+)
- **Модули Qt**: Core, Widgets, gui
- **Среда разработки:** Qt Creator
- **Компилятор:** MinGW
- **ОС:** Windows
- **Сборка:** qmake (файл `.pro`)
